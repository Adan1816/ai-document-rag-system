# RAG Document Assistant

A Retrieval-Augmented Generation (RAG) based AI system that allows users to upload documents and ask questions based on their content.

## Features
- PDF document ingestion
- Text chunking and embedding
- Vector search using FAISS
- Context-aware question answering using LLMs

## Tech Stack
- Python
- LangChain
- FAISS
- OpenAI API
